NAME
    man — system reference manual

SYNOPSIS
    man <topic>        display manual page for <topic>
    man -k             list available manual pages (apropos)
    man boot           return to the boot screen

DESCRIPTION
    Queries the system reference manual. Manual pages describe
    the components, protocols, and interfaces of the VILLaiN
    architecture.

    Pages are issued as part of the HITL diagnostic package.
    Additional entries may become available as system state
    changes.

AVAILABLE PAGES
    fair            conscience agent of the Governance Dyad
    handshake       dialectical co-sign protocol
    localhost       this diagnostic surface
    man             this page
